Libraries and Public School Performance in Chicago

A group project investigating the correlation between school performance and proximity to public libraries in Chicago

This project was done for GEO 425 at Michigan State University. I gathered data on the locations of public libraries and public high schools in Chicago, along with performance attributes on how many students graduated from each school and different test scores. I also gathered road data in Chicago. In ArcGIS Pro, I cleaned the road data to only show traversable roads and calculated how long it would take to drive on each road using the speed limit and length attributes. I performed network analysis by assigning each public library as a facility and finding each library's service areas that were within 2 minute and five minute drives. The data was reformatted into GeoJSON files and imported in kepler.gl where each school was symbolized based on its four-year graduation rate. Schools that fell inside library service areas tended to have higher graduation rates, although there are some outliers and when charted, the relationship between distance to libraries and graduation rate is only slghtly negatively correlated. Further data would be needed such as school and library funding to determine if distance is really a factor in school performance or if other attributes like wealth affect performance more.

Hospital Distribution in Michigan

A research report investigating hospital distribution, financial stability, and health outcomes in Michigan Counties

This project was done as an honors opportunity for GEO 425 at Michigan State University. I chose to use GIS to look into public health data of Michigan's counties to identify areas that were lacking in healthcare and find correlations between access to hospitals and the life expectancy and number of hospital visits in each county. I also dived deeper into the financial status of each county by the percentage of people in poverty and the average household income. I used Hot Spot Analysis (Getis-Ord Gi*) in ArcGIS Pro to identify counties that had lower life expectancies or more citizens in poverty. My findings indicated wealthier counties had more hopsitals within them, but that population density played a bigger role in health outcomes than just proximity to a medical facility.

Total Solar Eclipse Duration

A web map showing the path and duration of the April 8th, 2024 solar eclipse

Data was obtained containing the moon's path over the United States, the cities that experienced the total solar eclipse, and the duration of the eclipse. The data was first cleaned up in ArcGIS Pro to keep only the ciites impacted by the eclipse. The data was converted into GeoJSON format and imported into kepler.gl to construct the web map.

A map showing the strength of WiFi signals on Michigan State University's Campus

Data was gathered containing wifi signals collected by past MSU students on North Campus. Grape Cluster Yield using LiDAR

A group project investigating if drones or handheld LiDAR systems are better at object identification.

This project was completed collaboratively for GEO 829 at Michigan State University. My group and I invesitagted the research question are grape clusters better counted by Unoccupied Aerial Vehicles (UAVs) or by handheld Light Detection and Ranging (LiDAR) systems? We used CloudCompare to visually inspect the point clouds created using LiDAR which we then put through an unsupervised classification model to identify clusters of grapes. We then used Pix4D to to process the UAV data which was then also put through an unsupervised classification model. The UAV data also went through a deep learning model to identify objects/grape clusters. The results were preliminary and would require further investigation but LiDAR was better in this first stage of research.
